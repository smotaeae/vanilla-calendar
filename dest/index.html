VANILLA-CALENDAR DEMO

Vanilla js calendar without using additional packages.

Default

							
const calendar = new VanillaCalendar({
  calendar: document.querySelector('.vanilla-calendar'),
});

calendar.init();

Language and ISO 8601

The default language is Russian and ISO 8601 format, you can change it.

							
const calendar = new VanillaCalendar({
  calendar: document.querySelector('.vanilla-calendar'),
  settings: {
    lang: 'eng',
    iso8601: false,
  },
});

calendar.init();

Custom language

By default, only two languages ​​​​are available, these are Russian and English. You can add any language.

							
const calendar = new VanillaCalendar({
  calendar: document.querySelector('.vanilla-calendar'),
});

three.name.months.kz = ['Қаңтар', 'Ақпан', 'Наурыз', 'Сәуір', 'Мамыр', 'Маусым', 'Шілде', 'Тамыз', 'Қыркүйек', 'Қазан', 'Қараша', 'Желтоқсан'];
three.name.week.kz = ['Си', 'Же', 'Ду', 'Сй', 'Ср', 'Бе', 'Жұ'];
three.settings.lang = 'kz';

calendar.init();
