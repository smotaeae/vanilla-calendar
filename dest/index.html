VANILLA-CALENDAR

Vanilla js calendar without using additional packages.

GitHub 
							
const calendar = new VanillaCalendar({
  calendar: document.querySelector('.vanilla-calendar'),
});

calendar.init();

Language and ISO 8601

The default language is Russian and ISO 8601 format, you can change it.

							
const calendar = new VanillaCalendar({
  calendar: document.querySelector('.vanilla-calendar'),
  settings: {
    lang: 'eng',
    iso8601: false,
  },
});

calendar.init();

Custom language

By default, only two languages ​​​​are available, these are Russian and English. You can add any language.

							
const calendar = new VanillaCalendar({
  calendar: document.querySelector('.vanilla-calendar'),
});

calendar.name.months.kz = ['Қаңтар', 'Ақпан', 'Наурыз', 'Сәуір', 'Мамыр', 'Маусым', 'Шілде', 'Тамыз', 'Қыркүйек', 'Қазан', 'Қараша', 'Желтоқсан'];
calendar.name.week.kz = ['Си', 'Же', 'Ду', 'Сй', 'Ср', 'Бе', 'Жұ'];
calendar.settings.lang = 'kz';

calendar.init();

Highlight of the selected day

By default, on a click on a day, an event handler is triggered. This behavior can be disabled.

							
const calendar = new VanillaCalendar({
  calendar: document.querySelector('.vanilla-calendar'),
  settings: {
    selecting: false,
  },
});

calendar.init();

Highlight weekend and today

You can disable highlighting the weekend and today.

							
const calendar = new VanillaCalendar({
  calendar: document.querySelector('.vanilla-calendar'),
  settings: {
    weekend: false,
    today: false,
  },
});

calendar.init();

Displayed day, month, year

You can explicitly specify today, the selected date, month and year.

							
const calendar = new VanillaCalendar({
  calendar: document.querySelector('.vanilla-calendar'),
  today: new Date('2022-01-07'),
  settings: {
    selected: {
      date: '2022-01-09',
      month: 1,
      year: 2022,
    },
  },
});

calendar.init();

Date range

You can set the range of displayed dates, as well as specify which dates should be forcibly active.

							
const calendar = new VanillaCalendar({
  calendar: document.querySelector('.vanilla-calendar'),
  today: new Date('2022-01-08'),
  settings: {
    range: {
      min: '2022-01-07',
      max: '2022-02-12',
      values: ['2022-01-02', '2022-01-04'],
    }
  },
});

calendar.init();

More weekends or holidays

Easily specify your own weekend or holidays.

							
const calendar = new VanillaCalendar({
  calendar: document.querySelector('.vanilla-calendar'),
  today: new Date('2022-01-08'),
  settings: {
    selected: {
      month: 1,
      year: 2022,
      holidays: ['2022-01-03', '2022-01-04', '2022-01-05', '2022-01-20'],
    },
  },
});

calendar.init();

Do not show non-active months

Sometimes there is no need to display months that are not in the specified range.

							
const calendar = new VanillaCalendar({
  calendar: document.querySelector('.vanilla-calendar'),
  today: new Date('2022-02-08'),
  settings: {
    selected: {
      month: 2,
      year: 2022,
    },
    range: {
      min: '2022-01-07',
      max: '2022-02-12',
    },
    visibility: {
      months: false,
    },
  },
});

calendar.init();

Disable the display of arrows and year

Disable what you don't need.

							
const calendar = new VanillaCalendar({
  calendar: document.querySelector('.vanilla-calendar'),
  settings: {
    visibility: {
      year: false,
      arrows: {
        prev: false,
        next: false,
      },
    },
  },
});

calendar.init();
